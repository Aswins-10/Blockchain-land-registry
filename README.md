# Blockchain Land Registry System

A web-based land registry application built using **Python and Django** with basic blockchain concepts to ensure **secure and tamper-proof property records**.

## Features
- User registration and login  
- Land record management  
- Role-based access (Admin, User)  
- Secure data storage  
- Simple and clean interface  

## Tech Stack
- Python  
- Django  
- HTML, CSS, JavaScript  
- SQLite / MySQL  

## How to Run
```bash
git clone https://github.com/Aswins-10/Blockchain-land-registry.git
cd Blockchain-land-registry
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
